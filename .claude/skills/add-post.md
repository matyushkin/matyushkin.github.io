# Добавление новой публикации на страницу

Этот skill помогает добавить новую публикацию в список постов на странице posts/index.html.

## Что нужно сделать:

1. **Получить информацию о публикации** от пользователя:
   - Название публикации
   - Ссылка на публикацию
   - Дата публикации (формат: YYYY-MM-DD, например "2024-01-15")
   - Категория (Python, Data Science, Глубокое обучение, Головоломка, Математика, Общее)
   - Тип публикации (статья, перевод, подборка, тест, курс, головоломка, инструкция)
   - Количество просмотров (если известно, иначе "nan")
   - Рейтинг (если известен, от 1 до 5, иначе "nan")

2. **Открыть файл posts/index.html** и найти соответствующую категорию публикации.

3. **Добавить новую публикацию** в начало списка нужной категории в формате:
```html
<li data-rating="рейтинг" data-type="тип" data-views="просмотры">
    <a href="ссылка">
        <time datetime="дата">Название публикации</time>
    </a>
</li>
```

4. **Сохранить изменения** и уведомить пользователя об успешном добавлении.

## Категории публикаций:

- **Python** - публикации о языке Python
- **Data Science** - материалы по анализу данных и машинному обучению
- **Глубокое обучение** - статьи о нейросетях и deep learning
- **Головоломка** - алгоритмические задачи и головоломки
- **Математика** - материалы по математике
- **Общее** - публикации по другим IT-темам

## Типы публикаций:

- статья
- перевод
- подборка
- тест
- курс
- головоломка
- инструкция
- перевод

## Примеры:

Пример публикации:
```html
<li data-rating="5.0" data-type="статья" data-views="12345.0">
    <a href="https://proglib.io/p/python-decorators-guide-2024-01-15">
        <time datetime="2024-01-15">Полное руководство по декораторам в Python</time>
    </a>
</li>
```

Пример публикации без просмотров:
```html
<li data-rating="nan" data-type="перевод" data-views="nan">
    <a href="https://proglib.io/p/async-python-2024-01-20">
        <time datetime="2024-01-20">Асинхронное программирование в Python: руководство для начинающих</time>
    </a>
</li>
```

## Примечания:
- Новые публикации добавляются в начало соответствующего списка
- Если категория не существует, нужно создать новую секцию с заголовком `<h1>Название категории</h1>` и списком `<ul></ul>`
- Формат даты в атрибуте datetime: YYYY-MM-DD
- Если просмотры или рейтинг неизвестны, используйте "nan"
- Рейтинг должен быть числом от 1.0 до 5.0 или "nan"
