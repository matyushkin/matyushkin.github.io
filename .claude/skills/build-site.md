# Сборка сайта через Eleventy

Этот skill помогает собрать сайт через Eleventy, который минифицирует HTML-файлы.

## Что нужно сделать:

1. **Установить зависимости**, если они еще не установлены:
```bash
npm install
```

2. **Запустить сборку** через Eleventy:
```bash
npx eleventy
```

3. **Проверить результаты сборки**:
   - Убедиться, что в директории `_site/` появились минифицированные HTML-файлы
   - Проверить отсутствие ошибок в выводе команды

4. **Уведомить пользователя** о результатах сборки:
   - Успешно ли прошла сборка
   - Какие файлы были обработаны
   - Есть ли какие-либо предупреждения или ошибки

## Структура проекта:

- `.eleventy.js` - конфигурация Eleventy
- `_site/` - директория с минифицированными файлами (результат сборки)
- Исходные HTML-файлы находятся в корне и поддиректориях

## Процесс сборки:

Eleventy:
1. Читает конфигурацию из `.eleventy.js`
2. Находит все HTML-файлы
3. Минифицирует их с помощью html-minifier
4. Сохраняет результат в директорию `_site/`

## Опции минификации:

Согласно `.eleventy.js`, используются следующие опции:
- `useShortDoctype: true` - использовать короткий doctype
- `removeComments: true` - удалять комментарии
- `collapseWhitespace: true` - удалять лишние пробелы

## Примечания:

- После сборки файлы в `_site/` готовы к деплою
- Исходные файлы остаются без изменений
- Если сборка не удалась, проверьте корректность HTML в исходных файлах
- Перед деплоем всегда запускайте сборку, чтобы минифицировать изменения
