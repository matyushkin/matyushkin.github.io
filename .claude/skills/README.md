# Claude Code Skills для личной страницы matyushkin.github.io

Эта директория содержит набор skills (навыков) для Claude Code, которые помогают автоматизировать работу с личной страницей на GitHub Pages.

## Что такое Claude Code Skills?

Claude Code Skills - это специальные инструкции для AI-ассистента Claude, которые помогают выполнять типовые задачи по работе с проектом. Каждый skill содержит подробное описание задачи и шаги для её выполнения.

## Доступные Skills

### 1. **add-event.md** - Добавление IT-мероприятий
Помогает добавить новое IT-мероприятие на страницу событий (events/index.html).

**Использование:**
- Запросите у Claude: "Добавь новое событие"
- Claude запросит необходимую информацию (название, дата, ссылки, теги и т.д.)
- Событие будет добавлено в правильном формате

### 2. **add-post.md** - Добавление публикаций
Помогает добавить новую публикацию в список постов (posts/index.html).

**Использование:**
- Запросите: "Добавь новую публикацию"
- Укажите название, ссылку, категорию, тип и дату
- Публикация будет добавлена в соответствующую категорию

### 3. **update-links.md** - Управление ссылками
Помогает добавлять, обновлять или удалять ссылки на личные профили (links/index.html).

**Использование:**
- "Добавь ссылку на мой профиль в [платформа]"
- "Обнови ссылку на [платформа]"
- "Удали ссылку на [платформа]"

### 4. **update-cv.md** - Обновление резюме
Помогает обновить информацию в резюме (cv/index.html).

**Использование:**
- "Добавь опыт работы в [компания]"
- "Добавь навык [навык]"
- "Обнови образование"

### 5. **build-site.md** - Сборка сайта
Помогает собрать сайт через Eleventy для минификации HTML.

**Использование:**
- "Собери сайт"
- Claude запустит Eleventy и проверит результат

### 6. **deploy.md** - Деплой на GitHub Pages
Помогает задеплоить изменения на GitHub Pages.

**Использование:**
- "Задеплой изменения"
- Claude соберет сайт, создаст коммит и запушит изменения

## Как использовать Skills?

1. **Прямой запрос**: Просто попросите Claude выполнить нужную задачу, например:
   - "Добавь новое событие"
   - "Обнови резюме"
   - "Задеплой сайт"

2. **Claude автоматически выберет нужный skill** и запросит недостающую информацию.

3. **Следуйте инструкциям Claude** и предоставьте необходимые данные.

## Примеры использования

### Добавление нового события
```
Вы: Добавь новое событие про Python-митап
Claude: Отлично! Мне нужна следующая информация...
```

### Добавление публикации
```
Вы: Добавь статью "Декораторы в Python" на proglib.io
Claude: Хорошо! Укажите, пожалуйста:
- Дата публикации
- Тип (статья/перевод)
- Категория (Python)
```

### Деплой изменений
```
Вы: Задеплой изменения
Claude: Собираю сайт через Eleventy...
Claude: Создаю коммит...
Claude: Пушу изменения...
Claude: Готово! Сайт будет обновлен через 1-3 минуты.
```

## Расширение Skills

Вы можете создать свои собственные skills, добавив новые `.md` файлы в эту директорию. Каждый skill должен содержать:

1. **Описание задачи** - что делает этот skill
2. **Шаги выполнения** - пошаговая инструкция
3. **Примеры** - примеры использования и форматы данных
4. **Примечания** - важные замечания и советы
