# Добавление нового события на страницу

Этот skill помогает добавить новое IT-мероприятие в список событий на странице events/index.html.

## Что нужно сделать:

1. **Получить информацию о событии** от пользователя:
   - Название события
   - Дата и время (формат: "Day, DD Month в HH:MM", например "Tue, 24 November в 20:00")
   - Ссылка на страницу события
   - Ссылка на регистрацию
   - Описание события
   - Место проведения (Online, Москва, Санкт-Петербург и т.д.)
   - Месяц проведения (Январь, Февраль и т.д.)
   - Цена (0 для бесплатных, или сумма с валютой)
   - Теги (через запятую, например: "Python, Data Science")
   - Тип мероприятия (Митап, Конференция, Вебинар и т.д.)
   - Спикеры (опционально)
   - Компании (опционально)

2. **Открыть файл events/index.html** и найти место для вставки нового события.
   События должны быть отсортированы по дате (самые ранние вверху).

3. **Добавить новое событие** в формате:
```html
<section>
<div class="event"
     data-companies="Компании через пробел"
     data-loc="Место"
     data-month="Месяц"
     data-online="Online или пусто"
     data-price="Цена"
     data-tags="Теги через запятую"
     data-types="Тип">
    <h2>Название события</h2>
    <p class="time_and_space">
        <a href="ссылка-на-событие"><time>Дата и время</time></a>,
        <a href="ссылка-на-регистрацию">регистрация</a>.
    </p>
    <p class="description">Описание</p>
    <p class="themes_and_speakers">👥 Спикеры.</p>
</div>
</section>
```

4. **Сохранить изменения** и уведомить пользователя об успешном добавлении.

## Примеры:

Пример события:
```html
<section>
<div class="event"
     data-companies=""
     data-loc="Online"
     data-month="Декабрь"
     data-online="Online"
     data-price="0"
     data-tags="Python, Machine Learning"
     data-types="Вебинар">
    <h2>Введение в машинное обучение с Python</h2>
    <p class="time_and_space">
        <a href="https://example.com/event"><time>Mon, 15 December в 18:00</time></a>,
        <a href="https://example.com/register">регистрация</a>.
    </p>
    <p class="description">Вебинар о применении машинного обучения в Python с использованием библиотеки scikit-learn.</p>
    <p class="themes_and_speakers">👥 Иван Иванов, Петр Петров.</p>
</div>
</section>
```

## Примечания:
- Если событие бесплатное, data-price должен быть "0"
- Если место проведения - Online, то data-online должен быть "Online"
- Если спикеры не указаны, можно опустить блок `<p class="themes_and_speakers">`
- Теги должны быть разделены запятой и пробелом
