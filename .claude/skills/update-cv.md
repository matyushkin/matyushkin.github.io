# Обновление резюме (CV)

Этот skill помогает обновить информацию в резюме на странице cv/index.html.

## Что нужно сделать:

1. **Определить, что нужно обновить**:
   - Добавить новый опыт работы
   - Добавить новое образование
   - Добавить новый навык
   - Обновить контактную информацию
   - Добавить проект
   - Обновить описание

2. **Открыть файл cv/index.html** и найти соответствующий раздел.

3. **Внести изменения** в нужный раздел.

4. **Сохранить изменения** и уведомить пользователя.

## Основные разделы резюме:

- **Контактная информация** - email, телефон, ссылки на профили
- **Образование** - учебные заведения, степени, даты
- **Опыт работы** - компании, должности, обязанности, даты
- **Навыки** - технические и мягкие навыки
- **Проекты** - личные и профессиональные проекты
- **Публикации** - статьи, книги, курсы
- **Языки** - знание иностранных языков
- **Достижения** - награды, сертификаты

## Формат добавления опыта работы:

```html
<section>
    <h3>Должность в Компании</h3>
    <p class="period">Месяц Год - Месяц Год (или "По настоящее время")</p>
    <ul>
        <li>Обязанность или достижение 1</li>
        <li>Обязанность или достижение 2</li>
        <li>Обязанность или достижение 3</li>
    </ul>
</section>
```

## Формат добавления образования:

```html
<section>
    <h3>Название учебного заведения</h3>
    <p>Степень, Специализация</p>
    <p class="period">Год поступления - Год окончания</p>
</section>
```

## Формат добавления навыка:

```html
<li>Название навыка</li>
```

или с уточнением:

```html
<li>Категория: навык1, навык2, навык3</li>
```

## Формат добавления проекта:

```html
<section>
    <h3><a href="ссылка-на-проект">Название проекта</a></h3>
    <p>Краткое описание проекта и использованных технологий.</p>
</section>
```

## Примеры:

### Добавление опыта работы:

```html
<section>
    <h3>Senior Python Developer в Tech Company</h3>
    <p class="period">Январь 2022 - По настоящее время</p>
    <ul>
        <li>Разработка и поддержка backend-сервисов на Python/Django</li>
        <li>Оптимизация производительности приложений</li>
        <li>Менторство junior-разработчиков</li>
        <li>Участие в code review и архитектурных решениях</li>
    </ul>
</section>
```

### Добавление навыков:

```html
<ul>
    <li>Языки программирования: Python, JavaScript, TypeScript</li>
    <li>Фреймворки: Django, Flask, React, Node.js</li>
    <li>Базы данных: PostgreSQL, MongoDB, Redis</li>
    <li>Инструменты: Git, Docker, Kubernetes, CI/CD</li>
</ul>
```

## Примечания:

- Опыт работы обычно указывается в обратном хронологическом порядке (самый свежий сверху)
- Будьте конкретны в описании обязанностей и достижений
- Используйте глаголы действия: разработал, внедрил, оптимизировал
- Указывайте конкретные технологии и инструменты
- Добавляйте метрики и результаты, где это возможно
