# Обновление ссылок на личной странице

Этот skill помогает добавлять, обновлять или удалять ссылки на личной странице links/index.html.

## Что нужно сделать:

1. **Определить тип операции**:
   - Добавление новой ссылки
   - Обновление существующей ссылки
   - Удаление ссылки
   - Добавление новой категории

2. **Получить информацию от пользователя**:
   - Категория (или название новой категории)
   - Название ссылки
   - URL ссылки
   - Тип операции (добавить/обновить/удалить)

3. **Открыть файл links/index.html** и выполнить нужную операцию.

4. **Сохранить изменения** и уведомить пользователя.

## Категории ссылок:

- **💬 Социальные сети** - профили в соцсетях
- **👨‍💻️ Программирование** - профили на площадках для разработчиков
- **👨‍🔬️ Научные исследования** - научные профили
- **👨‍🏫️ Образовательная деятельность** - курсы и публикации
- **👨‍🎨️ Дизайн** - дизайнерские портфолио
- **📜 Стихи, рассказы, музыка, фильмы** - творческие профили
- **👨‍🎓️ Профили образовательных сайтов** - онлайн-курсы
- **👨‍💼️ Профили с резюме** - профили фрилансера

## Формат ссылки:

```html
<li>Название: <a href="url">текст ссылки</a></li>
```

или

```html
<li><a href="url">Название ссылки</a></li>
```

## Добавление новой ссылки:

Найдите соответствующую категорию (тег `<h2>`) и добавьте новый элемент `<li>` в список `<ul>` после заголовка.

## Добавление новой категории:

Создайте новый раздел с заголовком `<h2>` и списком `<ul>`:

```html
<h2>🔧 Новая категория</h2>
<ul>
    <li><a href="url">Название ссылки</a></li>
</ul>
```

## Обновление ссылки:

Найдите существующую ссылку и замените URL или текст.

## Удаление ссылки:

Найдите элемент `<li>` с нужной ссылкой и удалите его.

## Примеры:

### Добавление ссылки в существующую категорию:

В категорию "Социальные сети" добавить Telegram:
```html
<li>Telegram: <a href="https://t.me/username">t.me/username</a></li>
```

### Создание новой категории:

```html
<h2>🎮 Игры</h2>
<ul>
    <li>Steam: <a href="https://steamcommunity.com/id/username">steamcommunity.com/id/username</a></li>
    <li>Discord: <a href="https://discord.gg/invite">discord.gg/invite</a></li>
</ul>
```

## Примечания:

- Эмодзи в заголовках категорий помогают визуально различать секции
- Порядок категорий можно менять по усмотрению
- Предпочтительный формат: короткий домен вместо полного URL в тексте ссылки
- Убедитесь, что все ссылки работают и ведут на правильные страницы
