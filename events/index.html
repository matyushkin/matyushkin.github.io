Бонус для участников: возможность первыми поучаствовать в тестировании новых сервисов Облака.

  • Приветствие модератора
  • Новости Serverless
  • Интернет вещей в Яндекс.Облаке
  • Алертинг в Yandex Monitoring
  • Terraform-провайдер для Yandex Message Queue
Докладывают: Антон Черноусов, Глеб Борисов, Александр Сурков, Андрей Столбовский.
